JDBC

用JDBC连接一个数据库

特点：JDBC是面向“与平台无关”设计的
原理：JDBC为我们提供了一个“驱动程序管理器”，它能动态维护数据库查询所需的所有驱动程序对象.驱动程序对象会在装载时由“驱动程序管理器”自动注册，并可用Class.forName()强行装载。

加载数据库驱动

Class.forName()将对应的驱动类加载到内存中，然后执行内存中的static静态代码段，代码中，会创建一个驱动Driver的实例，放入DriverManager中，供DriverManager使用

当要执行的SQL语句有参数时，应怎么样使用statement

String url = "jdbc:mysql://localhost:3306/JDBCTEST";  
//设置值系统变量jdbc.drivers  
System.setProperty("jdbc.drivers", "com.mysql.jdbc.Driver");  
//2.通过特定的url获取driver  
Driver driver = DriverManager.getDriver(url);  
//打印是否存在  
System.out.println(driver);  
String uid="root";
String password="yuan1286";
Connection conn=DriverManager.getConnection(url,uid,password);

String sql="select * from USERS";
PreparedStatement ps=conn.prepareStatement(sql);
ResultSet rs=ps.executeQuery();
while(rs.next()){
    String username=rs.getString("username");
    String passwd=rs.getString("passwd");
    System.out.println(username+"    "+passwd);
}
rs.close();
ps.close();
conn.close();

try{
    conn.setAutoCommit(false);
    String sql="update USERS set passwd = '123456' where username = 'caixueyuan' and passwd= '12345678' ";
    PreparedStatement ps=conn.prepareStatement(sql);
    int rs=ps.executeUpdate();
    System.out.println(rs);
    conn.commit();
}
catch(SQLException e){
    conn.rollback();
    e.printStackTrace();
}

try{
    String username="hello";
    CallableStatement proc=conn.prepareCall("{ call select3(?,?)}");
    proc.setString("username", username);
    proc.registerOutParameter(2, Types.INTEGER);

    proc.execute();

    Integer number=proc.getInt("number");
    System.out.println(number.toString());
}
catch(SQLException e){
    e.printStackTrace();
}
