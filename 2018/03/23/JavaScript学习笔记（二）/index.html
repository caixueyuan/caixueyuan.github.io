JavaScript学习笔记（二）

发表于 2018-03-23 | 分类于 JavaScript学习笔记

表达式和运算符

算术运算符

在JavaScript中，所有的数字都是浮点型的，除法运算的结果也是浮点型的

NaN和其他任何值都是不相等的，包括它本身，通过x！==x来判断x是否为NaN，只有在x为NaN的时候，这个表达式的值才为true。

逻辑表达式

假值是false，null，undefined，0，-0，NaN和’’，所有的其他的值包括所有的对象都是真值

eval允许我们执行那些对上下文没有任何依赖的全局脚本代码段

delete不能够删除var声明的变量，返回false，在严格条件下，将抛出异常

var声明的全局变量是无法通过delete删除的。

语句

  • 由于对每个case的匹配操作实际上是“====”恒等运算符比较，而不是“==”相等运算符比较，所以，表达式和case的匹配并不会做任何类型转换。

  • for/in 循环并不会遍历对象的所有属性，只有“可枚举”的属性才能够遍历，由JavaScript语言核心所定义的内置方法不是“可枚举”的。

判断JavaScript实现是否支持严格模式

对象：JavaScript对象是动态的

属性特征：

  • 可写：标明是否可用设置该属性的值
  • 可枚举：标明是否通过for/in循环返回该属性
  • 可配置：表明是否能够删除或者修改该属性

对象

  • 包含属性
  • 对象的原型指向另外一个对象，本对象的属性继承自它的原型对象
  • 对象的类是一个标识对象类型的字符串
  • 对象的扩展标记指明了是否可以向该对象添加新属性

判断属性

  • hasOwnProperty()用来检测给定的名字是否是对象的自有属性
  • propertyIsEnumerable()只有检测是自有属性且这个属性的可枚举性为true
  • ！=== 判断一个属性是否为undefined
  • in 区分值为null的情况

Object.getOwnPropertyDescrptor()得到自由属性的描述符
Object.definePeoperty() 传入要修改的对象、要创建或者修改的属性的名称以及属性描述符对象

Object.getPrototypeOf()查询原型
Object.isPrototypeOf()检测一个对象是否是另一个对象的原型

Object.esExtensible()来判断该对象是否是可扩展的
Object.preventExtensions()将对象转换为不可扩展的，一旦设置为不可扩展，就没有办法设置为可扩展啦
Object.seal()将对象设置为不可扩展，将对象的所有的自有属性都设置为不可配置的
Object.isSealed()检测对象是否封闭
Object.freeze()将对象设置为不可扩展的和将其属性设置为不可配置的，将自有的所有数据属性设置为制只读
Object.isFrozen()检测对象是否冻结

数组

所有的索引都是属性名，但只有在0-2^32-2之间的整数属性名才是索引。本质上数组和对象之间是没有区别的，只是数组的索引都是0-2^32-2，而且自动维持length属性值的变化

push()在数组末尾增加一个或者多个元素
unshift()在数组的首部插入一个元素，并且将其他元素依次移到更高的索引处
pop()一次减少长度1并返回被删除元素
unshift()从头部删除一个元素，并将其他元素依次移动更低的索引处
splice()删除，删除，替换数组元素
Array.join()将数组中所有的元素都转化为字符串并连接在一起
数组.forEach(function(v,i,a){})从头到为遍历一遍数组，参数含义分别为（数组元素，元素的索引，数组的本身)

函数

函数调用
定义并调用一个函数来确定当前脚本运行时是否为严格模式

标识符arguments是指向实参对象的引用，实参对象是一个类数组对象

callee属性指代当前正在执行的函数

caller:调用当前正在执行的函数的函数

嵌套的函数不会将作用域内的私有成员复制一份，也不会对所绑定的变量生成静态快照

call() 和 apply()
call()的第一个参数都会变成this的值，哪怕传入的实参是原始值甚至是null或者undefined。
f.call(o,1,2);
f.apply(o,[1,2])
f为函数名，o为对象名字

bind()
将函数绑定至某个对象中

函数体代码的编译总是会在顶层函数中。

  • map对每个元素都进行操作，单独
  • reduce将多个元素都结合成一个元素

memorize()函数创建一个新的对象，这个对象呗当做缓存并赋值给一个局部变量，因此对于返回的函数来说它是私有的，所返回的函数将它的是实参数组转换为字符串，并将字符串用作缓存对象的属性名。如果在缓存中存在这个值，则直接返回它。
